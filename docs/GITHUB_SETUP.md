# GitHub Repository Setup (Recruiter Visibility)

Complete these steps on [github.com/Amrit000-HX/MediSense](https://github.com/Amrit000-HX/MediSense) to maximize profile impact.

## 1. Repository About section

Click the **gear** next to "About" on the repo home page:

| Field | Suggested value |
|-------|-----------------|
| **Description** | AI health companion: PDF medical report analysis, voice symptoms, 20+ languages — React, TypeScript, Express |
| **Website** | Your live demo URL (add after deploying) |
| **Topics** | `react` `typescript` `vite` `express` `healthcare` `health-tech` `fullstack` `i18n` `pdf` `portfolio` `hackathon` `nodejs` `tailwindcss` |

## 2. Pin on your profile

1. Go to https://github.com/Amrit000-HX
2. **Customize your pins** → select **MediSense**

## 3. Add screenshots

Follow [screenshots/README.md](./screenshots/README.md), commit images, and push.

## 4. Deploy a live demo (high impact)

| Part | Suggested platform |
|------|-------------------|
| Frontend (`dist/`) | [Vercel](https://vercel.com) or Netlify |
| API | [Railway](https://railway.app) or Render |

Set `VITE_API_URL` to your deployed API URL in the frontend host's environment variables.

Add the live URL to README under a **Live Demo** badge.

## 5. Social preview image

**Settings → General → Social preview** — upload a 1280×640 image (logo + tagline).

## 6. Enable GitHub Actions

After push, the **CI** workflow runs `npm run build` on every PR — shows a green checkmark to recruiters.

## 7. Star & share

- Star your own repo (shows activity)
- Link MediSense on LinkedIn / resume with 1-line impact statement from [PORTFOLIO.md](./PORTFOLIO.md)
